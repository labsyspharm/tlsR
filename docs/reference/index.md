# Package index

## TLS Detection

Identify tertiary lymphoid structures in spatial imaging data.

- [`detect_TLS()`](https://amiryousefilab.github.io/tlsR/reference/detect_TLS.md)
  : Detect Tertiary Lymphoid Structures using a KNN-density approach

## T-cell Clusters

Detect tumour-infiltrating T-cell clusters outside TLS.

- [`detect_tic()`](https://amiryousefilab.github.io/tlsR/reference/detect_tic.md)
  : Detect T-cell Immune Clusters (TIC) on the Tissue

## Spatial Scanning

Sliding-window Ripley’s L clustering map.

- [`scan_clustering()`](https://amiryousefilab.github.io/tlsR/reference/scan_clustering.md)
  : Scan Tissue for Local Immune Cell Clustering (Ripley's L Heatmap)

## Characterisation

Score and summarise detected structures.

- [`calc_icat()`](https://amiryousefilab.github.io/tlsR/reference/calc_icat.md)
  : Calculate ICAT (Independent Component Analysis Trace) Index for TLS
- [`summarize_TLS()`](https://amiryousefilab.github.io/tlsR/reference/summarize_TLS.md)
  : Summarize Detected TLS Across Samples

## Visualisation

Publication-ready spatial plots.

- [`plot_TLS()`](https://amiryousefilab.github.io/tlsR/reference/plot_TLS.md)
  : Plot Spatial Map of TLS and T-cell Clusters
